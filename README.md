AI Voice Assistant (Python)

* Description

This is a simple AI-based voice assistant built using Python.
It listens to voice commands and performs tasks like opening websites, telling time/date, and responding with speech.

* Features

- Speech recognition
- Text-to-speech response
- Open YouTube and Google
- Tell current time and date
- GUI with Start/Stop buttons

* Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- Tkinter

▶️ How to Run

1. Install libraries:
pip install speechrecognition pyttsx3 pyaudio

2. Run the program:
python your_file_name.py
